HOW TO CREATE YOUR FIRST 7-DAY SKU:
================================================================================

STEP 1: Create Vendor & Ingredients
------------------------------------
- Go to "Vendor Management"
- Click "Create Vendor"
- Add vendor details
- Add ingredients (Almonds, Walnuts, Cashews, etc.)
- Click "Save Vendor"

STEP 2: Create SKU with 7-Day Recipes
--------------------------------------
- Go to "SKU Management"
- Click "Create SKU"

Basic Info:
- Name: "Night Pack"
- Description: "Evening wellness blend"
- Target Weight: 44 grams per sachet
- Click "Next: Build Day Recipes"

Monday Recipe (üîµ Blue tab):
- Select Almonds ‚Üí Enter 6 grams ‚Üí Click "Add to MON Recipe"
- Select Walnuts ‚Üí Enter 5 grams ‚Üí Click "Add to MON Recipe"
- Select Cashews ‚Üí Enter 8 grams ‚Üí Click "Add to MON Recipe"
- ... add more ingredients until you reach 44g total
- Watch the weight indicator turn green!

Tuesday Recipe (üü£ Purple tab):
- Click "Next Day" or click TUE tab
- Add DIFFERENT ingredients/quantities for Tuesday
- Example: Almonds 7g, Walnuts 6g, Cashews 7g...
- Each day can be completely different!

Repeat for all 7 days:
- Wednesday, Thursday, Friday, Saturday, Sunday
- Each day gets its own unique recipe
- Progress shows "7/7 Complete" when done

Click "Save SKU" when all 7 days are complete!

STEP 3: Calculate Production Requirements
------------------------------------------
- Click "Production Calculator"
- Select: "Night Pack"
- Select: "Monthly Pack (28 sachets)"
- Enter: 80 packs
- Click "Calculate Requirements"

You'll see:
- Total sachets: 2,240 (80 packs √ó 28 sachets)
- Multiplier: 320 (80 packs √ó 4 weeks)
- Each day needs 320 sachets
- Consolidated ingredient totals
- Day-by-day breakdown (expandable)
- Vendor recommendations
- Shortage alerts (in red)

Export Options:
- Click "Export CSV" ‚Üí Share with procurement team
- Click "Print" ‚Üí Print production order

STEP 4: Set Pricing
-------------------
- Go to "Pricing Strategy"
- Select your SKU and pack type
- Enter costs (packaging, operating, marketing, etc.)
- Set profit margin or enter selling price
- Save pricing for both Weekly and Monthly packs

STEP 5: Plan Sales Targets
---------------------------
- Go to "Sales & Revenue"
- Select month and year
- Add target rows for your SKU
- Set quantity targets
- View projections and break-even analysis

PRODUCTION CALCULATION EXAMPLE:
================================================================================

Scenario: Produce 80 Monthly Packs of Night Pack

Calculation:
- Monthly Pack = 4 Weekly Packs
- Multiplier = 80 √ó 4 = 320
- Need 320 of EACH day's sachet

If Monday Recipe has:
- Almonds: 6g per sachet
- Walnuts: 5g per sachet
- Cashews: 8g per sachet

Then for Monday:
- Almonds: 320 √ó 6g = 1,920g = 1.92 kg
- Walnuts: 320 √ó 5g = 1,600g = 1.6 kg
- Cashews: 320 √ó 8g = 2,560g = 2.56 kg

If Tuesday Recipe has:
- Almonds: 7g per sachet
- Walnuts: 6g per sachet
- Cashews: 7g per sachet

Then for Tuesday:
- Almonds: 320 √ó 7g = 2,240g = 2.24 kg
- Walnuts: 320 √ó 6g = 1,920g = 1.92 kg
- Cashews: 320 √ó 7g = 2,240g = 2.24 kg

... Repeat for all 7 days ...

Consolidated Totals:
- Total Almonds = MON + TUE + WED + THU + FRI + SAT + SUN
- Total Walnuts = MON + TUE + WED + THU + FRI + SAT + SUN

TROUBLESHOOTING:
================================================================================

Issue: "npm: command not found"
Solution: Install Node.js from https://nodejs.org/

Issue: "Missing script: dev"
Solution: Make sure you're in the correct folder:
cd "D:\POKAsoft\wkly nuts app\wkly nuts app\vendor and product management tool"
Then run: npm install

Issue: PowerShell script blocked
Solution: Run PowerShell as Administrator:
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

Issue: Port already in use
Solution: App will use next available port (5174, 5175, etc.)
Check terminal output for actual URL

Issue: White/blank screen
Solution: Clear browser cache, reload page
